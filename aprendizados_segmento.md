# Aprendizados de Segmento - Qualificação MQL

## Varejo B2C (Micro/Pequeno Porte)
- **Data:** 2025-06-25
- **Lead:** Temap Store / Luciano Junior
- **Decisão:** Não-MQL

### Perfil do Segmento
- **Modelo:** Varejo B2C (e-commerce + loja física)
- **Porte:** Micro (R$250k-500k faturamento, 1-10 pessoas)
- **ERP:** Bling (adequado para este porte)
- **Operação:** Venda direta para consumidor final

### Por que não é ICP T1
1. **Posição na cadeia errada:** Não é fornecedor para revendas/lojistas
2. **Falta de objeto B2B:** Não há tabela de preço B2B, catálogo para revenda ou pedido recorrente
3. **Porte incompatível:** Volume incompatível com operação atacadista de alto giro
4. **ERP já adequado:** O Bling já cobre as necessidades de gestão deste porte

### Sinais de Alerta para Desqualificação Rápida
- Faturamento abaixo de R$1 milhão/ano
- Operação B2C (não B2B)
- Menos de 10 funcionários
- Ausência de menção a vendas para outras empresas/revendedores

### Próximos Passos
- Manter este perfil para nutrição (caso haja crescimento futuro)
- Monitorar evolução de porte (R$1M+ de faturamento)
- Avaliar novamente se houver mudança para modelo B2B
